\# Q-Learning Maze Solver



An interactive Python application that uses \*\*Reinforcement Learning (Q-Learning)\*\* to find the shortest path through a maze. 



\## Features

\* \*\*Q-Learning Algorithm\*\*: Trains an agent to navigate obstacles.

\* \*\*Tkinter GUI\*\*: Visualizes the agent (blue circle) moving through the maze.

\* \*\*Comparison\*\*: Watch the difference between random exploration and the learned optimal path.
